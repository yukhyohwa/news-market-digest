# 如何运行脚本

您可以通过命令行参数来控制文章抓取的时间范围。

## 1. 默认行为 (抓取过去1天的文章)

不带任何参数运行脚本，将抓取从昨天到现在的文章。

```bash
python main.py
```

## 2. 按天数抓取

使用 `--days` 参数指定要抓取过去多少天的文章。

例如，抓取过去7天的文章:
```bash
python main.py --days 7
```

## 3. 按指定日期范围抓取

使用 `--range` 参数指定一个明确的开始和结束日期。

**格式为 `YYYYMMDD-YYYYMMDD`**。

例如，抓取从2025年12月16日到2025年12月17日的文章:
```bash
python main.py --range 20251216-20251217
```

**注意**: 当使用 `--range` 参数时，`--days` 参数会被忽略。
